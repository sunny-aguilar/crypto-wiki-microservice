REST API Docs

Base URL

http://flipX.engr.oregonstate.edu:55055/

To connect to the Wiki scraper API, you must be logged on to OSU VPN and use the base URL as the starting point. This is where GET requests can be sent along with parameters to obtained the desired information.

URL Parameters

URL parameters are passed via the URL to the endpoint to specify the data to be obtained from the Wiki scraping service. An example of the usage where you are searching for a term and want the Wiki scraper to return the URL of the page as well as a page summary, your HTTP request would look like this:

http://localhost:55055/api?search_term=dog&title=on&summary=on
